# ST Market Intelligence — Vision

## 1. الرؤية

بناء منصة احترافية مستقلة لتحليل سوق الأسهم والخيارات الأمريكية، تحول البيانات المعقدة إلى معطيات واضحة، سهلة، وقابلة لاتخاذ القرار.

## 2. الهدف

مساعدة المتداول على فهم:

- اتجاه السهم.
- قوة الفرصة.
- جودة السيولة.
- تأثير القاما.
- الزخم الحالي.
- جودة العقد.
- نقاط الدخول والإلغاء والأهداف.
- سبب النتيجة بلغة مفهومة.

## 3. الجمهور المستهدف

المنصة تخدم:

- المتداول المبتدئ.
- المتداول المتوسط.
- المتداول المحترف.

ويتم عرض المعلومات على ثلاث طبقات:

1. نتيجة مختصرة وسهلة.
2. تفاصيل تحليلية متوسطة.
3. بيانات عميقة للمحترف.

## 4. المشكلة التي تحلها المنصة

معظم منصات السوق تعرض أرقامًا ومؤشرات كثيرة، لكن المستخدم قد لا يعرف:

- ما المهم منها؟
- ماذا تعني؟
- هل توجد فرصة فعلًا؟
- متى يدخل؟
- متى يلغي السيناريو؟

المنصة لا تكتفي بعرض البيانات، بل تفسرها وتحولها إلى نتيجة عملية.

## 5. الميزة الأساسية

كل سهم يحصل على تحليل موحد يشمل:

- السعر.
- الاتجاه.
- القاما.
- السيولة.
- الزخم.
- السوق العام.
- جودة العقد.
- درجة الفرصة.
- درجة الثقة.
- تفسير النتيجة.

## 6. مبادئ المشروع

- المشروع مستقل بالكامل عن جميع المشاريع والبوتات القديمة.
- لا يتم نسخ أي كود أو قاعدة بيانات أو منطق قديم.
- يستخدم المشروع Massive API لبيانات الخيارات.
- يستخدم Finnhub API لأسعار الأسهم.
- سهولة الاستخدام لا تعني تقليل جودة المعلومات.
- لا تظهر أي معلومة بلا قيمة عملية.
- يجب التحقق من جودة وحداثة البيانات قبل التحليل.
- لا يتم تحويل كل حالة إلى توصية شراء أو بيع.
- يجب توضيح المخاطر ودرجة الثقة.
- يجب أن تكون جميع الوحدات قابلة للاختبار بشكل مستقل.

## 7. تجربة المستخدم

يجب أن يستطيع المستخدم خلال ثوانٍ معرفة:

- هل السهم يستحق المتابعة؟
- هل الفرصة صاعدة أم هابطة؟
- هل الحالة مراقبة أم جاهزة أم مفعلة؟
- ما سبب النتيجة؟
- أين مستوى إلغاء السيناريو؟
- ما أفضل عقد إذا كانت الفرصة مناسبة؟

## 8. حالات التحليل

- NEUTRAL — محايد
- WATCH — مراقبة
- READY — جاهزة
- ACTIVE — مفعلة
- WEAKENING — تضعف
- INVALIDATED — ملغاة

## 9. معيار الجودة

نجاح المنصة لا يقاس بعدد المؤشرات، بل بقدرتها على:

- تقديم بيانات صحيحة وحديثة.
- تقليل الإشارات الضعيفة.
- توضيح أسباب النتيجة.
- تسجيل الأداء وقياسه.
- تحسين النتائج بناءً على الاختبارات.
- مساعدة المستخدم على اتخاذ قرار أكثر انضباطًا.

## 10. النتيجة النهائية المستهدفة

منصة سهلة للمبتدئ، عميقة للمحترف، وتقدم تحليلًا واضحًا بدل إغراق المستخدم بالأرقام.
